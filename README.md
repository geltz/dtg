a simple tkinter gui that generates [Danbooru](https://danbooru.donmai.us/) tags using [dart-v2-moe-sft](https://huggingface.co/p1atdev/dart-v2-moe-sft). it contains encoded lists of [artist](https://danbooru.donmai.us/tags?commit=Search&search%5Bcategory%5D=1&search%5Bhide_empty%5D=yes&search%5Border%5D=count), [copyright](https://danbooru.donmai.us/tags?commit=Search&search%5Bcategory%5D=3&search%5Bhide_empty%5D=yes&search%5Border%5D=count), [character](https://danbooru.donmai.us/tags?commit=Search&search%5Bcategory%5D=4&search%5Bhide_empty%5D=yes&search%5Border%5D=count), [meta](https://danbooru.donmai.us/tags?commit=Search&search%5Bcategory%5D=5&search%5Bhide_empty%5D=yes&search%5Border%5D=count) and [general](https://danbooru.donmai.us/tags?commit=Search&search%5Bcategory%5D=0&search%5Bhide_empty%5D=yes&search%5Border%5D=count) tags.

**keybinds**

| input | action |
| :--- | :--- |
| enter | generate |
| ctrl+c | copy tags |
